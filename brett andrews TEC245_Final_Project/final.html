Welcome to TriniTing before we start please fill out the required information then click Start



Rules of the game are as follows

 1)When a question is correct the screen would flash green

 2)When a question is incorrect the screen would flash red

 3)A player will be challenged to maintain an answering streak, if they answer three consecutive questions incorrectly their game must be terminated

 4)A player wins the game if they complete the selected category by answering all questions correctly

 5)A player receives a nice try if they complete the selected category but had some questions incorrect/exit the game
